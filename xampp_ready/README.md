# Smart Planner untuk XAMPP

Folder ini adalah versi yang siap dipakai di XAMPP.

## Cara pasang
1. Salin folder `xampp_ready` ke folder `htdocs` di XAMPP.
2. Jalankan Apache dari XAMPP Control Panel.
3. Buka browser ke:
   - http://localhost/xampp_ready/

## Catatan
- Aplikasi menggunakan SQLite yang disimpan di `data/smart_planner.sqlite`.
- File database akan otomatis dibuat saat pertama kali halaman dibuka.
- Jika ingin menambah fitur, Anda bisa lanjutkan dari file `dashboard.php` dan `new_roadmap.php`.
